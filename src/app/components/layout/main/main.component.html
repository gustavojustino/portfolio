Projetos em destaque

Carro API

Este projeto fullstack implementa operações CRUD utilizando Java com Spring Boot no backend e TypeScript com Angular no frontend. Utiliza MySQL como banco de dados para armazenamento. A autenticação e autorização são gerenciadas pelo Spring Security, integrado com tokens JWT para controle seguro de acesso.

Petshop System

O sistema de Petshop adota uma arquitetura distribuída com várias tecnologias interconectadas. A autenticação inicial é feita por uma aplicação Java, garantindo segurança no acesso. A aplicação principal do Petshop é desenvolvida em Go, aproveitando sua eficiência e desempenho. A integração entre as partes do sistema assegura funcionalidade eficiente no gerenciamento das operações do Petshop.

Verificador de idade

Este projeto web simples foi criado com HTML, CSS e JavaScript. Ele exibe a idade e o gênero do usuário de maneira clara e intuitiva. A interface é projetada para oferecer uma experiência direta, permitindo ao usuário acessar facilmente as informações apresentadas sem dificuldades adicionais.
